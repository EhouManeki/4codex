# X投稿用スライド構成（全8枚）

- 00: 表紙（テーマ + 7項目一覧 + 保存CTA）
- 01: ① ファーストビューに「数字」を入れる
- 02: ② CTAボタンは「動詞」で始める
- 03: ③ スマホ表示を先に設計する
- 04: ④ フォームの項目は最小限（3つ以下）に絞る
- 05: ⑤ お客様の声は「変化前→変化後」で書く
- 06: ⑥ 価格の前に「価値」を伝える
- 07: ⑦ 「読まれる」前提ではなく「見られる」前提で設計する

各スライド共通:
- サイズ: 1080 x 1350（X投稿向け）
- 構成: 見出し + 実務で使える要点2行 + 保存を促すフッター
- トーン: ダーク背景 + アクセント色で視認性重視
